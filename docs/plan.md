# 小红书卡片与文案生成系统 - 项目执行计划

## 项目概述

基于 React + Express + AI 的小红书卡片与文案生成系统，支持用户输入产品描述后自动生成符合小红书风格的文案和精美卡片图。

**技术栈**：

- 前端：React 18 + TypeScript + Vite + Tailwind CSS
- 后端：Express + TypeScript + Vercel AI SDK
- AI 服务：OpenAI GPT-4o / Claude 3
- 部署：Vercel (前端) + Railway/Render (后端)
